# RojgarSetu: Local Job Portal System

**"Connecting local talent with local opportunities — digitally."**

---

## 🌍 Background
In the Koshi Province of Nepal (Dharan, Itahari, Biratnagar), many youths migrate to Kathmandu or abroad seeking work, unaware of the growing opportunities within their own local cities. **RojgarSetu** (Bridge to Employment) acts as a digital bridge to connect local employers with qualified local workers, eliminating agency fees and reducing unnecessary migration.

## 🏢 About the System
RojgarSetu is a modern, web-based platform specifically designed for the **Koshi Province**. It provides a direct connection between job seekers and businesses, fostering economic growth within the local community.

## ✨ Key Features
*   **Job Search:** Category, location, and salary filters to find relevant local jobs.
*   **Apply Online:** Directly apply for positions through the system.
*   **Application Tracking:** Real-time status updates for all applied jobs.
*   **Employer Dashboard:** Tools for businesses to post jobs and manage applicants.
*   **Admin Reports:** Detailed insights on job trends and monthly statistics for local governance.

## 👨‍💼 User Roles
1.  **Job Seekers:** Local individuals like Ramesh from Itahari who want to find work close to home.
2.  **Employers:** Local business owners like Sunita Didi from Dharan who need to hire specialized staff.
3.  **Admin:** Municipality or ward-level staff who verify accounts and manage system safety.

## 🔄 How It Works
1.  **Registration:** Employer registers and waits for Admin approval.
2.  **Posting:** Approved employers post job openings which are then reviewed by Admin.
3.  **Applying:** Job seekers browse approved listings and submit applications.
4.  **Selection:** Employers view profiles, select candidates, and update notification statuses.

## 🛠️ Tech Stack
*   **Frontend:** JSP, CSS3 (Modern Design System), JavaScript.
*   **Backend:** Java Servlets.
*   **Database:** MySQL.
*   **Tools:** IntelliJ IDEA, Tomcat Server.

## 🎯 System Objectives
*   Provide local job opportunities to the youth of Koshi Province.
*   Simplify the hiring process for local businesses.
*   Eliminate middlemen and high agency fees for direct connections.
*   Increase the employment rate in Koshi Province through a digital platform.

## 🏙️ Target Area
The system focuses on the major hubs of **Koshi Province**, including:
*   Dharan
*   Itahari
*   Biratnagar
*   Damak
*   Birtamod

### 🚀 Getting Started
1. Clone the repository.
2. Open the project in **IntelliJ IDEA**.
3. Configure **Tomcat Server** and set the Application Context to `/`.
4. Import the provided SQL schema into your MySQL database.
5. Run the project and access it at `http://localhost:8080/`.
